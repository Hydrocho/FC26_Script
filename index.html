모바일에서 열기 (QR 코드)

PC와 스마트폰이 같은 와이파이에 연결되어 있어야 합니다.

모바일 설치 방법
iOS (아이폰 / 사파리)
  1. App Store에서 Userscripts 확장 앱(무료)을 다운로드 및 활성화합니다.
  2. 본 웹앱에서 [설치 URL 복사] 버튼을 누릅니다.
  3. 사파리에서 Userscripts 확장 프로그램 아이콘을 누르고 "+" -> "Install from URL"을 선택합니다.
  4. 복사한 URL을 붙여넣고 확인을 누르면 설치 확인 창이 나타납니다. "Install"을 눌러 완료합니다.
  5. 사파리에서 EA FC 26 웹앱에 접속하면 자동으로 활성화됩니다.
iOS (오리온 브라우저)
  1. App Store에서 Chrome/Firefox 확장을 지원하는 Orion Browser를 설치합니다.
  2. Orion 브라우저 설정 -> Extensions -> Firefox / Chrome 웹스토어에서 Tampermonkey를 설치합니다.
  3. 본 웹앱에서 [설치 URL 복사] 버튼을 누릅니다.
  4. 새 탭을 열고 복사한 URL을 주소창에 붙여넣어 접속합니다.
  5. Tampermonkey 설치 페이지가 열리면 "설치 (Install)"를 누릅니다.
안드로이드 (키위 브라우저)
  1. Google Play Store에서 Kiwi Browser를 설치합니다.
  2. Chrome 웹스토어에 접속하여 Tampermonkey 확장 프로그램을 추가합니다.
  3. 본 웹앱에서 [설치 URL 복사] 버튼을 누릅니다.
  4. 새 탭을 열고 복사한 URL을 주소창에 붙여넣어 접속합니다.
  5. Tampermonkey 설치 페이지가 자동으로 활성화되면 "설치 (Install)"를 누릅니다.
